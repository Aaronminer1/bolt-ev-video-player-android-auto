Bolt Player

YouTube & web browser for your car's Android Auto display

⬇ Install

Tap Install above — your browser will download the APK.
Then tap the download notification to install.

If prompted during install

1
Tap Settings on the "blocked" warning
2
Toggle Allow from this source
3
Tap back, then tap Install
4
Open Android Auto → grant overlay permission when asked

DO NOT use while driving. This app is for use only when the vehicle is safely parked. Watching video while operating a motor vehicle is illegal in most jurisdictions and extremely dangerous.
